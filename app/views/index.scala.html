What is Snooker Orienteering

Snooker O is a combination of a standard Score event with the rules of Snooker.

In the main variant, a competitor must punch a 'red' control and then a 'colour' control until all the reds have been cleared. Then the colours need to be punched in the correct order for Snooker (Yellow, Green, Brown, Blue, Pink, Black).

Of course, all this is within the time limits of a Score event so there are penalties for being late.
