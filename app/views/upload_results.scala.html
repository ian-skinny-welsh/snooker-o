Upload your results file

To export your event results from SiTiming:

  • Use the up icon on the left to navigate to the top level of the event in SI
  • Select 'Results' from the menu
  • Select 'Export Results' from the menu
  • Select 'Processable CSV Results' from the menu
  • Select the folder to save to and set the file name
  • Ensure the 'Delimiter' is set to 'Comma'
  • Ensure the 'Encoding' is set to 'Western European (Windows)'
  • Select the 'Write file' button
